AI project · Claude Code skill

A tool that writes equity-research-grade competitive analysis

This is a skill I built for Claude Code — an instruction file that tells it exactly how to do one specific job well, instead of leaving it to however it would approach the question by default.

The problem

Competitive positioning work — mapping a company against its peers on cost, technology, pricing power, moat — is something I used to do by hand as an equity analyst. Done properly, it's slow: pull the filings, chase down earnings-call commentary, check review sites for product-quality signals, then write it up in a structure that actually supports a verdict rather than just describing things. Most AI-generated versions of this skip straight to the writing and skip the verdict — smooth paragraphs that never quite commit to an answer.

What I built

The skill forces the order of operations I'd insist on from a junior analyst: research before writing, always. It won't let Claude answer from memory — it has to run 15 to 30+ real searches across company filings, earnings calls, peer comparisons, and customer review sites before drafting a single sentence.

Then it writes to a fixed structure: industry economics first, company background second, then a company-by-peer verdict — ahead, at parity, or behind — across thirteen specific dimensions (cost position, pricing power, network effects, lock-in, and so on), each one backed by a cited fact, not an adjective. Output is always a formatted Word document, not a chat reply.

Seeing it work

I ran it live on Cloudflare against its closest peers — Akamai, Fastly, and AWS CloudFront — to see what it actually produces.

Headline finding: Cloudflare leads on most of the thirteen dimensions — platform breadth, a real data network effect from the traffic passing through it, and rising customer retention. But it's still not GAAP-profitable, and a May 2026 restructuring tied to an AI-first pivot got a rough market reaction the same quarter it posted record revenue. Not a clean win — a real, current tension.
